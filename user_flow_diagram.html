User Flow Description

Regular User Flow

The typical user journey through the Inkspire Library Management System follows these paths:

  1. Entry Point: Users begin at the Home Page, which showcases featured books, categories, and navigation options.
  2. Authentication: If not logged in, users are directed to the Login/Register page to create an account or sign in.
  3. Book Discovery: After authentication, users can browse books by category, search for specific titles, or explore featured collections.
  4. Book Details: Selecting a book displays its detailed information, including description, author details, availability, and reviews.
  5. Book Actions: From the book details page, users can:
    • Borrow the book (submit a borrow request)
    • Purchase the book
    • Add a review or rating
    • Add the book to their favorites
  6. User Profile: Users can access their profile to view:
    • Currently borrowed books
    • Borrowing history
    • Favorite books
    • Personal information
    • Notifications

Administrator Flow

Administrators have access to additional functionality:

  1. Admin Dashboard: After logging in as an admin, users can access the administrative dashboard.
  2. User Management: Admins can view, edit, and manage user accounts.
  3. Book Management: Admins can add new books, update existing ones, and manage inventory.
  4. Category Management: Admins can create and edit book categories.
  5. Borrow Request Management: Admins can approve or reject borrow requests and process returns.
  6. Reports and Analytics: Admins can view system statistics and generate reports.
